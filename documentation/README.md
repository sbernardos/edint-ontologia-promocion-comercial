# Documentación de la ontología

Esta carpeta contiene la **documentación orientada a personas**. 


# Propósito

El objetivo de esta carpeta es almacenar **documentación de la ontología orientada a las personas**, destinada a publicarse en la web y a facilitar la comprensión de la conceptualización y su reutilización por parte de terceros.

Esta documentación facilita la **reutilización y el mantenimiento** de la ontología.

# Contenidos
Include here any files  used to publish the ontology, such as:
Incluir cualquier fichero utilizado para publicar la ontología en la web, como por ejemplo:

- **Ficheros HTML** — Generados usando[Widoco](https://github.com/dgarijo/Widoco). Estos fichero spueden incluir ejemplos de uso de la ontología, consultas SPARQL, enlaces restricciones en SHACL, etc.


# Formatos aceptados

Pueden usarse formatos como:
- `.html` — Ficheros HTML   
- `.md`, `.pdf` — Informes o documentación sobre resultados de evaluación. 
- Ficheros adicionales necesarios para el contenido de la web en HTML. 

# Buenas prácticas
- Mantener los archivos de documentación versionados junto con las actualizaciones de la ontología.
